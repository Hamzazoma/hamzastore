# متجر ملابس مع لوحة إدارة

مشروع متجر عربي جاهز للنشر على `Netlify`، مبني بـ `Vite + TypeScript` مع:

- واجهة متجر مستوحاة من أسلوب المتاجر الكبيرة مثل الصورة المرفوعة
- أقسام: `منتخبات` و`رجال` و`سيدات` و`بنات` و`أطفال` و`جزم`
- اختيار مقاسات داخل المنتج
- سلة شراء ونموذج تأكيد طلب
- لوحة إدارة منفصلة لمتابعة الطلبات
- تحديث تلقائي للطلبات كل `10` ثوانٍ
- إضافة منتج جديد مع `صورة + سعر + مقاسات + قسم`
- تخزين فعلي للطلبات والمنتجات على `Netlify Blobs` بعد النشر

## الصفحات

- المتجر: `/`
- الإدارة: `/admin.html`
- مسار سهل للإدارة أيضًا: `/admin`

## التشغيل المحلي

1. تثبيت الحزم:

```bash
npm install
```

2. تشغيل الواجهة فقط:

```bash
npm run dev
```

عند التشغيل المحلي بـ `Vite` فقط، سيستخدم المشروع `localStorage` كنسخة تجريبية للطلبات والمنتجات.

## تجربة وظائف Netlify محليًا

إذا أردت تجربة الربط الحقيقي مع `Netlify Functions` محليًا:

```bash
npx netlify dev
```

## النشر على Netlify

1. ارفع المشروع إلى `GitHub`
2. اربط الريبو مع `Netlify`
3. الإعدادات:
   - Build command: `npm run build`
   - Publish directory: `dist`
4. سيقرأ `Netlify` إعدادات الدوال من ملف `netlify.toml`

## الملفات المهمة

- `index.html`: صفحة المتجر
- `admin.html`: صفحة لوحة الإدارة
- `src/main.ts`: منطق الواجهة الرئيسية
- `src/admin.ts`: منطق لوحة الإدارة
- `src/api.ts`: الربط بين الواجهة والتخزين
- `netlify/functions/shop.mjs`: API الخاصة بالطلبات والمنتجات
- `shared/default-data.mjs`: الأقسام والمنتجات المبدئية

## ملاحظات

- عند أول تشغيل على `Netlify` يتم زرع منتجات مبدئية تلقائيًا
- يمكن إضافة منتجات جديدة من لوحة الإدارة، وستظهر مباشرة في المتجر
- الطلبات الجديدة ستظهر في الإدارة وتتحدث تلقائيًا كل 10 ثوانٍ
